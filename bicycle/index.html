Personal Project

Motorized Bicycle Conversion

Integrated an internal combustion engine onto a bicycle frame with a custom mounting solution.

Overview

This project converted a standard bicycle frame into a motorized build by mounting an internal combustion engine directly to the frame. The frame wasn't designed to carry engine loads or drivetrain forces, so the core challenge was building a mounting solution that kept the frame structurally sound while keeping the new drivetrain properly aligned.

Highlights

  • Integrated an internal combustion engine onto a bicycle frame with custom mounting solutions to maintain structural integrity and drivetrain alignment.
